## How to download and play
1. Download the game on my [itch.io](https://thiagoianuch.itch.io/warehouse-wreckage) **(Windows only)**
2. Unzip the folder and run WarehouseWreckage.exe

**Controls:**
* WASD / Arrows - Move
* Space - Shoot
* R - Restart game
